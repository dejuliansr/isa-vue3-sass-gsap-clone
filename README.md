## Project Setup

### 1. Clone the Repository
```sh

```

```sh
npm install
```

### 3. Start Development Server

```sh
npm run dev
```

### 4. Build for Production
```sh
npm run build
```

### 5. Preview Production Build
```sh
npm run preview
```
